## 1. 월세 ↔ 전세 ↔ 매매 관계
```
┌─────────┐
│ 매매가  │
└────┬────┘
     │
   영향
     ↓
┌─────────┐    전환  ┌─────────┐
│ 전세가  │ ←────→   │  월세가 │
└─────────┘          └─────────┘
```
**금리가 높아지면**
- 전세 → 월세 전환 ↑(집주인이 전세금을 은행에 맡기면 이자 수익 ↑)
- 월세 수요 ↑ → 월세 가격 ↑
```
금리가 오르면 집주인은 전세보증금을 은행에 맡겼을 때 받는 **무위험 수익률(risk-free return)**이 증가합니다.

따라서:
전세보증금 1억원 → 은행 예치 시 이자 400만원
월세 환산 가치 = 400만원 / 12 = 월 33만원

즉, 월 33만원보다 적은 월세를 받는 것은 경제적으로 손해입니다.
그래서 월세를 올리게 되는 구조가 됩니다.

이것이 바로 **(전세보증금 × 금리 = 월세 환산값)**이라는 개념입니다.

즉, 금리(이자율)가 월세의 기준점 역할을 합니다.
```
**금리가 낮아지면**
- 전세 → 월세 전환 ↓ (은행 이자가 낮아서 전세 선호)
- 월세 수요 ↓ → 월세 가격 ↓


## 2. 기준 금리
**모든 금리의 기준점**
기준금리↑ → 대출 이자 부담↑ → 매매 수요↓ → 전세 수요↑ → 월세 전환↑


## 3. 전세자금대출 금리
세입자가 전세 보증금을 마련할 때 받는 대추 금리 부동산 영향:
- 전세자금대출 금리↑ → 전세 부담↑ → 월세로 이동
- 전세자금대출 금리↓ → 전세 선호↑ → 월세 수요↓

## 4. 변동형주택담보대출 금리
- 주택담보대출 금리 ↑ → 집 사기 어려움 → 임대 수요 ↑
- 집주인도 대출 이자 부담 → 월세 인상 압박

## 5. 무담보콜금리, KORIBOR, CD금리
- 전세보증금의 기회비용 측정
- 집주인이 전세보증금 1억을 받으면:
    - CD금리 3.5% → 은행 예금 시 연 350만원 수익
    - CD금리 2.5% → 은행 예금 시 연 250만원 수익
- 금리가 낮으면 전세보다 월세가 유리하지 않음

## 6. 기업대출 금리
- 기업대출 금리 ↑ → 경기 둔화 예상 → 부동산 수요 ↓
- 다주택자, 부동산 투자자들의 투자 의사결정 지표

## 7. 소비자물가
- 물가 ↑ → 실질 임대료 가치 ↓ → 명목 월세 인상 압박
- 물가가 2.4% 오르면, 같은 월세 100만원의 실질 가치는 97.6만원
- 집주인이 이를 방어하려고 월세 인상

## 8. Target : 환산보증금_평당가
```
df["적용이자율"] = (df["기준금리"] + 2.0) / 100.0
df["환산보증금(만원)"] = (df["보증금(만원)"] + (df["임대료(만원)"] * 12)) / df["적용이자율"]
df["환산보증금_평당가"] = df["환산보증금(만원)"] / df["전용평수"]
```
