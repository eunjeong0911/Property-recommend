## 2. 타겟 변수 (Target)

### 2.1 베이지안 보정 성사율
```python
베이지안_성사율 = (m × Prior + 거래완료) / (m + 총매물수)
```

| 변수 | 값 | 설명 |
|------|-----|------|
| `m` | **91** | 베이지안 임계값 (중위 매물 수) |
| `Prior` | 지역별 평균 | 지역 평균 성사율 (가중치 적용) |

### 2.2 3등급 분류 (Quantile 기준)

| 등급 | 숫자값 | 기준 (베이지안 성사율) | 개수 | 비율 |
|------|--------|------------------------|------|------|
| **하위 (Bottom)** | 0 | < 0.6576 | 112 | 33% |
| **중위 (Middle)** | 1 | 0.6576 ~ 0.7759 | 112 | 33% |
| **상위 (Top)** | 2 | ≥ 0.7759 | 115 | 34% |

> ✅ **균등 분포**: Quantile 기반 3분위로 클래스 불균형 해결

## 3. 피처 엔지니어링 (19개 피처)

### 3.1 피처 목록

| 카테고리 | 피처명 | 공식/설명 |
|----------|--------|-----------|
| **원본** | 직원수, 공인중개사, 중개보조원 | 원본 컬럼 |
| **비율** | 공인중개사비율, 중개보조원비율 | 자격자/보조원 비율 |
| **이진** | 복수자격, 전원자격, 대형, 중형, 소형 | 조건 충족 여부 |
| **경력** | 운영연수, 운영로그, 연차구간 | 등록일 기반 |
| **파생** | 전문밀도, 경력전문, 규모연차, 경력규모 | 복합 피처 |
| **비율2** | 보조비, 비자격비 | 비자격 인력 비중 |

### 3.2 Data Leakage 방지
| ❌ 제외된 피처 | 제외 사유 |
|----------------|-----------|
| ~~직원당거래~~ | `거래완료` 사용 → 타겟과 직접 관련 |
| ~~거래완료~~ | 타겟 계산에 사용 |
| ~~총매물수~~ | 타겟 계산에 사용 |

## 4. 모델 학습 (5-Model Ensemble)

### 4.1 사용 모델

| 모델 | 알고리즘 | 특징 |
|------|----------|------|
| **LightGBM** | Gradient Boosting | 빠른 학습, 정확도 |
| **XGBoost** | Gradient Boosting | 안정적, 정규화 |
| **RandomForest** | Bagging | 과적합 방지, 다양성 |
| **CatBoost** | Gradient Boosting | 범주형 강점, 정규화 |
| **SVM** | Support Vector | 소규모 데이터 강점 |

### 4.2 앙상블 구성
- **Voting Classifier** (Soft Voting)
- 5개 모델의 확률을 평균하여 최종 예측

### 4.3 하이퍼파라미터 최적화
- **RandomizedSearchCV** (30 iterations)
- **5-Fold StratifiedKFold** 교차검증
- **정규화**: `reg_alpha` (L1), `reg_lambda` (L2)

## 6. SHAP 피처 중요도 분석

### 6.1 Top 10 피처

| 순위 | 피처 | SHAP 값 | 해석 |
|------|------|---------|------|
| **1** | 연차구간 | 0.0895 | 경력 단계 (0~3) |
| **2** | 경력규모 | 0.0861 | 운영연수 × 직원수 |
| **3** | 운영연수 | 0.0852 | 사무소 경력 (년) |
| **4** | 직원수 | 0.0838 | 사무소 규모 |
| **5** | 경력전문 | 0.0762 | 운영연수 × 공인중개사비율 |
| 6 | 운영로그 | 0.0674 | log(운영일수) |
| 7 | 규모연차 | 0.0628 | log(직원수) × 운영연수 |
| 8 | 중개보조원 | 0.0436 | 보조원 수 |
| 9 | 공인중개사 | 0.0416 | 자격자 수 |
| 10 | 전문밀도 | 0.0393 | 공인중개사 / 직원수 |

### 6.2 핵심 인사이트

```
┌─────────────────────────────────────────────────────┐
│  "경력이 가장 중요하다"                             │
│                                                     │
│  연차구간(9%) + 경력규모(9%) + 운영연수(9%)         │
│  = 총 27%                                          │
│                                                     │
│  → 경력 관련 피처가 전체 중요도의 1/4 이상!         │
└─────────────────────────────────────────────────────┘
```

### 6.3 비즈니스 해석

| 발견 | 의미 |
|------|------|
| **경력 > 규모** | 오래 운영한 사무소가 더 신뢰됨 |
| **규모도 중요** | 직원수가 Top 4 피처 |
| **전문성 효과** | 공인중개사 비율이 높으면 긍정적 |
| **복합 피처 우수** | 경력×규모, 경력×전문성 조합이 효과적 |

## 11. 참고사항

### 11.1 Data Leakage 방지
- `거래완료`, `총매물수`는 피처에서 **완전 제외**
- `직원당거래` 피처도 제거 (간접 누수)

### 11.2 클래스 균형
- `class_weight="balanced"` 적용
- Quantile 기반 3분위로 균등 분할

### 11.3 정규화 적용
- L1 정규화 (`reg_alpha`): 0.0 ~ 1.0
- L2 정규화 (`reg_lambda`): 1.0 ~ 10.0
