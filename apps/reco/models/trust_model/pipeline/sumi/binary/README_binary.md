# 🏠 중개사 신뢰도 모델 - 이진 분류 파이프라인

부동산 중개사 사무소의 신뢰도를 **2등급 (상위/하위)**으로 예측하는 앙상블 머신러닝 파이프라인입니다.

---

## 📁 파일 구조

```
pipeline/binary/
├── _00_load_data.py           # 데이터 로드
├── _01_targer_engineering.py  # 타겟 변수 생성 (2등급)
├── _02_feature_engineering.py # 피처 엔지니어링 (19개 피처)
├── _03_train_model.py         # 5-Model 앙상블 학습
├── _04_predict.py             # 예측 수행
├── run_all.py                 # 전체 파이프라인 실행
└── README_binary.md           # 이 문서
```

---

## ⚡ 빠른 실행


## 2. 타겟 변수 (2등급 분류)

### 2.1 베이지안 보정 성사율
```python
베이지안_성사율 = (m × Prior + 거래완료) / (m + 총매물수)
```

| 변수 | 값 | 설명 |
|------|-----|------|
| `m` | **91** | 베이지안 임계값 |
| `Prior` | 지역별 평균 | 가중치 적용 |

### 2.2 2등급 분류 (중위수 기준 - 50/50)

| 등급 | 숫자값 | 기준 | 개수 | 비율 |
|------|--------|------|------|------|
| **하위** | 0 | < 0.7266 | 169 | **50%** |
| **상위** | 1 | ≥ 0.7266 | 170 | **50%** |

> ✅ **균등 분포**: 중위수 기준으로 정확히 50:50 분할

# 🆚 다중분류 vs 이진분류 비교

## 성능 비교

| **분류 기준** | 33%/66% Quantile | 50% 중위수 | 단순화 |
| **Test F1** | 48.5% | **53.1%** | **+4.6%p ↑** |
| **Test Accuracy** | 50.0% | **55.9%** | **+5.9%p ↑** |
| **랜덤 기준선** | 33% | 50% | - |
| **기준선 대비** | +15.5%p | **+5.9%p** | - |
| **CV F1 (최고)** | 46.5% (CatBoost) | **70.7%** (LightGBM) | **+24.2%p ↑** |
