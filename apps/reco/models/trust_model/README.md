# 🏗️ 중개사 신뢰도 예측 모델

## 📊 개요

거래 성사율 기반으로 중개사를 3등급(A, B, C)으로 분류하는 머신러닝 모델입니다.

- **모델 타입**: Voting Ensemble (Soft Voting)
- **Target**: 3등급 분류 (A: 우수, B: 보통, C: 미흡)
- **성능**: Test Accuracy 57.9%
- **특징**: Data Leakage 완전 제거, 실전 예측 가능

### Confusion Matrix
```

─────────────────────────
A (우수)      21    4    1
B (보통)      13   10    2
```

### 해석
- **A등급 탐지 우수**: 81% 정확도로 우수 중개사 추천 가능

### 2. 모델 로드 및 예측
```python
import pickle
import pandas as pd
import numpy as np

# 모델 로드
with open('apps/reco/models/trust_model/models/voting_ensemble.pkl', 'rb') as f:
    model_package = pickle.load(f)

ensemble = model_package['ensemble']
scaler = model_package['scaler']
label_encoder = model_package['label_encoder']
features = model_package['features']

# 새로운 데이터 예측
new_data = pd.DataFrame({
    '등록매물': [50],
    '영업일수': [1000],
    '영업년수': [2.7],
    # ... 나머지 15개 피처
})

# 스케일링
X_scaled = scaler.transform(new_data[features])

# 예측
pred_encoded = ensemble.predict(X_scaled)
pred_grade = label_encoder.inverse_transform(pred_encoded)
pred_proba = ensemble.predict_proba(X_scaled)

print(f"예측 등급: {pred_grade[0]}")
print(f"A등급 확률: {pred_proba[0][0]:.2%}")
print(f"B등급 확률: {pred_proba[0][1]:.2%}")
print(f"C등급 확률: {pred_proba[0][2]:.2%}")
```

## 🔍 주요 특징

### 1. Data Leakage 완전 제거
```
❌ 제외된 피처:
- 거래완료 (Target 계산에 직접 사용)
- 총매물수 = 등록매물 + 거래완료
- 거래성사율 (Target 자체)
- 구평균성사율, 지역권평균성사율 (거래완료 기반)

✅ 안전한 피처만 사용:
- 매물 정보, 시간 정보, 지역 정보, 안전성
```

### 2. Voting Ensemble
```
장점:
✅ 강한 모델만 사용 (RF, XGB, GB, ET)
✅ Soft Voting으로 확률 기반 예측
✅ 가중치로 강한 모델 강조 (RF=2, XGB=2)
✅ 단순하고 빠름 (Stacking보다)
```

### 3. 실전 활용 가능
```
용도:
✅ 우수 중개사 추천 (A등급 81% 정확도)
✅ 미흡 중개사 필터링 (C등급 52% 정확도)
✅ 확률 기반 신뢰도 제공
```
