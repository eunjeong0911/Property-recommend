# Next.js API Routes (`src/app/api`)

이 폴더는 **Next.js가 제공하는 백엔드 서버 기능(API Routes)**을 위한 공간입니다.

## 📌 역할
- 클라이언트(브라우저)가 요청을 보낼 수 있는 **실제 URL 엔드포인트**를 만듭니다.
- 이 폴더 안의 파일들은 **Node.js 서버 환경**에서 실행됩니다.

## 📂 예시
- `auth/[...nextauth]/route.ts`: NextAuth.js를 이용한 로그인/로그아웃 처리 엔드포인트 (`/api/auth/...`)
- 데이터베이스에 직접 접근하거나, 비밀 키를 다뤄야 하는 로직이 있다면 여기에 작성합니다.

---
**비유**: "우리 집 현관문" (외부에서 들어오는 요청을 받아 처리하는 곳)
