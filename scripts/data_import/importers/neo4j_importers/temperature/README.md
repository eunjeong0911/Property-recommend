# 🌡️ 온도 점수 산정 시스템

매물의 특성을 다양한 카테고리로 분석하여 **36.5도 기준** 온도 점수로 변환하는 시스템입니다.

---

## 📊 온도 카테고리 요약

| 카테고리 | 평가 항목 | 기준 데이터 |
|----------|-----------|-------------|
| 🔒 **안전 온도** | 범죄율, CCTV, 경찰/소방서 | 5대 범죄 현황, 안전시설 |
| 🚇 **교통 온도** | 지하철 접근성, 업무지구, 버스 | 지하철 승하차 데이터 |
| 🏪 **생활 온도** | 편의점, 세탁소, 마트 | 편의시설 거리 |
| 🎨 **문화 온도** | 영화관, 미술관, 도서관, 공원 | 문화시설 거리 |
| 🐾 **펫 온도** | 놀이터, 동물병원, 공원, 펫샵 | 반려동물 시설 |

## 🔒 안전 온도 (Safety Temperature)

### 점수 구성 (총 100점)

| 항목 | 비중 | 설명 |
|------|------|------|
| 지역 치안 점수 | 30% | 자치구별 5대 범죄 발생률 기반 |
| 골목 안전 점수 | 20% | 반경 200m 내 CCTV·비상벨 개수 |
| 응급 인프라 점수 | 50% | 경찰서·소방서까지의 거리 |

### 사용 데이터

- **5대 범죄 발생현황** (살인, 강간, 강도, 절도, 폭력)
- **CCTV 위치 데이터** (서울시 공공데이터)
- **비상벨 위치 데이터** (서울시 공공데이터)
- **경찰서/소방서 위치 데이터**

### 세부 계산식

```
[지역 치안]
Risk = (살인×3.0) + (강간×2.0) + (강도×1.8) + (폭력×1.0) + (절도×0.7)
Score = (1 - 정규화된 Risk) × 100

[골목 안전]
Score = Min(100, CCTV×5 + 비상벨×2)

[응급 인프라]
경찰 점수 = Max(0, (1000m - 거리) / 10)
소방 점수 = Max(0, (2500m - 거리) / 25)
Score = (경찰 점수 + 소방 점수) / 2
```

## 🚇 교통 온도 (Traffic Temperature)

### 점수 구성 (총 100점)

| 항목 | 배점 | 설명 |
|------|------|------|
| 역 등급 점수 | 40점 | 가장 가까운 지하철역의 등급 |
| 업무지구 접근성 | 30점 | 주요 업무지구까지의 거리 |
| 버스 점수 | 30점 | 반경 300m 내 버스정류장 수 |

### 사용 데이터

- **지하철 승하차 인원 정보** (호선별/역별/시간대별)
- **지하철역 위치 데이터**
- **버스정류장 위치 데이터**

### 역 등급 산정 기준

| 등급 | 조건 |
|------|------|
| **S** | 점수 ≥ 90 (4개 노선 이상 환승역) |
| **A** | 점수 ≥ 70 (3개 노선 환승역) |
| **B** | 점수 ≥ 60 (2개 노선 환승역) |
| **C** | 점수 ≥ 40 (단일 노선) |
| **D** | 점수 < 40 |

### 주요 업무지구 목록

```
가산디지털단지, 서울역, 여의도, 선릉, 시청, 
강남, 역삼, 잠실, 삼성(무역센터), 을지로입구
```

## 🏪 생활 온도 (Convenience Temperature)

### 점수 구성 (총 80점 → 100점 스케일링)

| 항목 | 배점 | 최적 거리 | 0점 거리 |
|------|------|-----------|----------|
| 편의점 | 20점 | ≤ 50m | ≥ 150m |
| 세탁소 | 20점 | ≤ 100m | ≥ 250m |
| 마트/백화점 | 40점 | ≤ 300m | ≥ 1000m |

### 사용 데이터

- **편의점 위치 데이터** (GS25, CU, 세븐일레븐 등)
- **세탁소 위치 데이터**
- **대형마트/백화점 위치 데이터**

### 거리 기반 점수 계산

```
Score = Max_Score × (1 - (거리 - 최적거리) / (0점거리 - 최적거리))
```

### 사용 데이터

- **문화시설 위치 데이터** (영화관, 미술관, 공연장, 박물관, 도서관, 문화원)
- **공원 위치 데이터**

### 거리 감쇠 공식

```
Score = Weight × (1 - 거리/500m)
// 500m 이내 시설만 점수에 반영
```

## 🔄 온도 변환 공식

모든 카테고리의 Raw Score(0~100)는 **36.5도 기준 온도**로 변환됩니다.

```python
# 글로벌 평균 계산
global_avg = mean(모든 매물의 raw_score)

# 온도 변환
if raw_score <= global_avg:
    temperature = raw_score × (36.5 / global_avg)
else:
    temperature = 36.5 + (raw_score - global_avg) × (63.5 / (100 - global_avg))
```

### 온도 해석

| 온도 범위 | 의미 |
|-----------|------|
| 36.5°C | 평균 |
| > 36.5°C | 평균 이상 (우수) |
| < 36.5°C | 평균 이하 |
| 100°C | 최상위 |
| 0°C | 최하위 |
