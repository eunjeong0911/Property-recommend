# 안전온도 분석 보고서

## 개요

서울시 25개 구의 안전 인프라(CCTV, 지구대/파출소, 소방서, 안전비상벨)와 범죄 발생 현황을 종합하여 지역별 안전온도를 산출했습니다. 안전온도는 0-100점 척도로 표현되며, 점수가 높을수록 안전한 지역입니다.

## 데이터 수집

### 사용한 데이터
1. **5대 범죄 발생현황** (2024년)
   - 살인, 강도, 강간·강제추행, 절도, 폭력
   - 발생 건수 및 검거 건수

2. **CCTV 정보**
   - 서울시 전체 CCTV 57,699대
   - 설치목적구분: 생활방범, 차량방범, 어린이보호 등
   - 안전 관련 CCTV만 선별하여 분석

3. **지구대/파출소 정보**
   - 서울시 지구대 및 파출소 위치
   - 구별 경찰 인프라 분포

4. **소방서 정보**
   - 서울시 25개 소방서
   - 구별 소방 인프라 분포

5. **안전비상벨 정보**
   - 총 49개 설치
   - 일부 구에만 설치되어 있음

## 분석 방법론

### 1. 범죄 분석
- **구별 범죄 발생 현황**: 2024년 5대 범죄 총 발생 건수
- **범죄 유형별 분석**: 살인, 강도, 강간·강제추행, 절도, 폭력 각각의 발생 및 검거 현황
- **검거율 분석**: 범죄 유형별 평균 검거율 계산

### 2. 안전 인프라 분석
- **CCTV**: 생활방범, 차량방범, 어린이보호 목적의 안전 CCTV만 집계
- **지구대/파출소**: 구별 경찰 인프라 개수
- **소방서**: 구별 소방서 개수 (대부분 구당 1개)
- **안전비상벨**: 구별 안전비상벨 개수

### 3. 안전온도 계산

**정규화**
- Min-Max Scaler를 사용하여 모든 지표를 0-100 범위로 변환
- 범죄 발생은 역수로 계산 (적을수록 높은 점수)
- 안전 인프라는 많을수록 높은 점수

**가중 평균**
```
안전온도 = (범죄 점수 × 0.30) + (CCTV 점수 × 0.30) + 
          (지구대/파출소 점수 × 0.20) + (소방서 점수 × 0.10) + 
          (안전비상벨 점수 × 0.10)
```

**등급 분류**
- 매우안전: 66 ≤ 온도 ≤ 100
- 안전: 33 ≤ 온도 < 66
- 주의: 0 ≤ 온도 < 33

## 주요 발견사항

### 범죄 현황
- 서울시 전체 25개 구 분석 완료
- 범죄 발생이 많은 지역과 적은 지역의 편차가 큼
- 검거율은 범죄 유형에 따라 차이가 있음

### 안전 인프라
- **CCTV**: 양천구, 강북구, 도봉구 등이 안전 CCTV가 많음
- **지구대/파출소**: 구별로 고르게 분포
- **소방서**: 구당 1개씩 배치
- **안전비상벨**: 일부 구에만 설치되어 있어 형평성 문제

### 안전온도 결과
- 25개 구의 안전온도를 0-100점으로 산출
- 범죄가 적고 안전 인프라가 많은 지역이 높은 점수
- 등급별 분포를 통해 안전 수준 파악
