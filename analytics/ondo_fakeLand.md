# 허위매물 온도 분석 보고서

## 개요

서울시 25개 구의 부동산 중개업소 현황, 행정처분 이력, 경매 통계를 종합하여 지역별 허위매물 위험도를 정량화한 '허위매물 온도'를 산출했습니다. 허위매물 온도는 0-100점 척도로 표현되며, 점수가 높을수록 허위매물 위험이 높은 지역입니다.

## 데이터 수집

### 사용한 데이터
1. **서울시 부동산 중개업소 정보**
   - 총 25,450개 중개업소
   - 영업 상태: 영업중, 휴업, 업무정지, 휴업연장
   - 주소, 전화번호, 조회수 등

2. **행정처분 데이터** (2017-2024년)
   - 총 3,355건
   - 등록취소, 업무정지, 적발건수
   - 연도별, 분기별, 구별 통계

3. **지역별 매각통계 (경매)**
   - 25개 구의 부동산 경매 건수
   - 경매는 시장 불안정성의 지표

## 분석 방법론

### 1. 중개업소 분석
- **구별 중개업소 수**: 부동산 거래 활성도 지표
- **영업 상태 분석**: 영업중, 휴업, 업무정지, 휴업연장
- **문제업소 비율**: (휴업 + 업무정지 + 휴업연장) / 전체 중개업소 × 100

### 2. 행정처분 분석
- **행정처분 점수 계산**:
  - 등록취소 = 5점 (가장 중대한 위반)
  - 업무정지 = 3점 (중간 수준 위반)
  - 적발건수 = 1점 (경미한 위반)
  
- **행정처분 비율**: 행정처분 점수 / 중개업소 수
  - 중개업소 수 대비 처분 정도를 정규화
  - 규모가 다른 지역 간 공정한 비교 가능

- **연도별 추이**: 2017-2024년 행정처분 변화
- **분기별 패턴**: 계절별 행정처분 경향

### 3. 경매 분석
- **구별 경매건수**: 부동산 시장 불안정성 지표
- **경매 vs 중개업소**: 거래 활성도와의 관계
- **경매 vs 행정처분**: 시장 불안과 위반의 상관관계

### 4. 허위매물 온도 계산

**정규화**
- Min-Max Scaler를 사용하여 모든 지표를 0-100 범위로 변환

**가중 평균**
```
허위매물 온도 = (행정처분 비율 × 0.40) + (경매건수 × 0.30) + 
               (문제업소 비율 × 0.30)
```

**가중치 근거**
- 행정처분 비율 40%: 직접적인 위반 행위 지표, 가장 중요
- 경매건수 30%: 시장 불안정성, 간접적 위험 지표
- 문제업소 비율 30%: 현재 영업 상태, 신뢰도 지표

**등급 분류**
- 고위험: 66 ≤ 온도 ≤ 100
- 주의: 33 ≤ 온도 < 66
- 안전: 0 ≤ 온도 < 33

## 주요 발견사항

### 중개업소 현황
- 총 25,450개 중개업소 중 99.6%가 영업중
- 휴업 86개, 업무정지 10개, 휴업연장 4개
- 구별 중개업소 수 편차가 큼 (부동산 거래 활성도 차이)

### 행정처분 현황
- 2017-2024년 총 3,355건의 행정처분
- 연도별 추이: 증가/감소 패턴 분석
- 분기별 패턴: 특정 분기에 집중되는 경향
- 구별 편차: 일부 지역에 행정처분이 집중

### 경매 현황
- 25개 구의 경매 건수 집계
- 경매가 많은 지역 = 시장 불안정 또는 거래 활발
- 중개업소 수와 양의 상관관계
- 행정처분과의 상관관계 분석

### 허위매물 온도 결과
- 25개 구의 허위매물 온도를 0-100점으로 산출
- 행정처분이 많고, 경매가 많고, 문제업소 비율이 높은 지역이 높은 점수
- 등급별 분포를 통해 위험 수준 파악

### 주요 상관관계

- **행정처분 vs 경매건수**: 상관관계 분석 결과 제시
- **문제업소 비율 vs 행정처분**: 위반 이력과 현재 상태의 관계
