## 🌡️ 교통온도 계산 시스템

### 📌 교통온도란?
부동산 매물의 **대중교통 접근성**을 **30~43°C 온도**로 표현한 지표
- **30°C (차가움)** = 대중교통 불편 (역세권 밖, 버스 적음)
- **36.5°C (평균 체온)** = 보통 수준 (일반 서울 주거지)
- **43°C (뜨거움)** = 대중교통 매우 편리 (역세권, 버스망 우수)

---

## 1단계: 지하철 점수 계산 (0~1점)

### 🚇 지하철 점수 = 역까지 거리 + 역의 중요도

#### A. 거리 점수 (Distance Score)
**얼마나 역에 가까운가?**

```
거리          점수     의미
─────────────────────────────────
0~250m       1.0점   🟢 1차 역세권 (도보 3분)
250~500m     0.5점   🟡 2차 역세권 (도보 6분)
500~1000m    0.18점  🟠 역세권 밖 (도보 12분)
2000m 이상   0.01점  🔴 매우 멀음
```

**계산 공식:**
- **0~250m**: 점수 = 1.0 (만점)
- **250~500m**: 점수 = 1.0 - 0.5 × (거리-250)/250 (선형 감소)
- **500m 이상**: 점수 = 0.5 × e^(-(거리-500)/500) (급격히 감소)

**예시:**
- 역에서 100m: 1.0점 (바로 앞)
- 역에서 375m: 0.75점 (중간)
- 역에서 1000m: 0.18점 (멀음)

---

#### B. 역 중요도 점수 (Station Quality Score)
**그 역이 얼마나 중요한가?**

```
역 점수 = (출퇴근 비율 × 40%) + (역 규모 × 20%)
```

**1) 출퇴근 비율 (40% 가중치)**
- 전체 승하차 중 출퇴근 시간대(07-09시, 18-20시) 비중
- 높을수록 = 직장인이 많이 이용 = 출퇴근 편의성 우수

```
역명              출퇴근비율   정규화 점수
────────────────────────────────────
가산디지털단지    46.1%       0.92점  (업무지구)
여의도            45.3%       0.90점  (금융가)
강남              30.2%       0.60점  (상업+업무)
주택가 역         15.0%       0.30점  (주거 위주)
```

**2) 역 규모 (20% 가중치)**
- 하루 전체 승하차 인원
- 많을수록 = 큰 역 = 환승역/주요역 가능성 높음

```
역명        일일 승하차      정규화 점수
─────────────────────────────────
강남역      300만명         1.0점
서울역      310만명         1.0점
작은 역     20만명          0.1점
```

**역 점수 예시:**
- 강남역: 0.6 × 0.4 + 1.0 × 0.2 = **0.44점**
- 주택가 작은 역: 0.3 × 0.4 + 0.1 × 0.2 = **0.14점**

---

#### C. 최종 지하철 점수
**가까운 역 3개를 거리에 따라 가중 평균**

```python
# 가까운 역 3개 선택
역1 = 양재역 (거리 300m, 역점수 0.3)
역2 = 매봉역 (거리 800m, 역점수 0.25)
역3 = 도곡역 (거리 1200m, 역점수 0.35)

# 각 역의 종합 점수 = 거리점수 × 역점수
역1 종합 = 0.8 × 0.3 = 0.24
역2 종합 = 0.2 × 0.25 = 0.05
역3 종합 = 0.08 × 0.35 = 0.028

# 거리 가중치 (가까울수록 중요)
역1 가중치 = 1/(1+300/250) = 0.45
역2 가중치 = 1/(1+800/250) = 0.24
역3 가중치 = 1/(1+1200/250) = 0.17

# 가중 평균
최종 점수 = (0.24×0.45 + 0.05×0.24 + 0.028×0.17) / (0.45+0.24+0.17)
         = 0.13점
```

## 2단계: 버스 점수 계산 (0~1점)

### 🚌 버스 점수 = 가장 가까운 정류장 거리 + 주변 정류장 밀도

#### A. 거리 점수 (70% 가중치)
**가장 가까운 정류장까지 얼마나 가까운가?**

```
거리 점수 = e^(-거리/300)
```

**예시:**
```
거리          점수     의미
─────────────────────────────
0m           1.00점   🟢 바로 앞
50m          0.85점   🟢 매우 가까움
150m         0.61점   🟡 가까움
300m         0.37점   🟠 조금 멀음
600m         0.14점   🔴 멀음
```

---

#### B. 밀도 점수 (30% 가중치)
**반경 300m 내에 정류장이 몇 개나 있는가?**

```
밀도 점수 = min(정류장 개수 / 8, 1.0)
```

**예시:**
```
정류장 개수    점수     의미
────────────────────────────
0개           0.0점   🔴 버스 없음
4개           0.5점   🟡 보통
8개           1.0점   🟢 우수
16개          1.0점   🟢 매우 우수 (상한선)
```

---

#### C. 최종 버스 점수

```python
# 예시: 양재동 어느 매물
가장_가까운_정류장 = 80m
정류장_개수_300m내 = 6개

# 거리 점수
거리_점수 = e^(-80/300) = 0.76

# 밀도 점수
밀도_점수 = min(6/8, 1.0) = 0.75

# 최종 버스 점수 (거리 70%, 밀도 30%)
버스_점수 = 0.7 × 0.76 + 0.3 × 0.75
         = 0.532 + 0.225
         = 0.757점
```

## 3단계: 지하철 vs 버스 통합 (교통 점수)

```
교통 점수 = 지하철 점수 × 55% + 버스 점수 × 45%
```
[교통점수 출처](https://www.newdaily.co.kr/site/data/html/2022/04/06/2022040600150.html#:~:text=2030%EC%84%B8%EB%8C%80%EA%B0%80%20%ED%86%B5%EA%B7%BC(%ED%86%B5%ED%95%99)%EC%97%90%20%EC%A3%BC%EB%A1%9C%20%EC%9D%B4%EC%9A%A9%ED%95%98%EB%8A%94%20%EA%B5%90%ED%86%B5%EC%88%98%EB%8B%A8%EC%9D%80%20%EC%A7%80%ED%95%98%EC%B2%A0(25.5%25)%2C%20%EB%B2%84%EC%8A%A4+%EC%A7%80%ED%95%98%EC%B2%A0(23.4%25)%2C,20%EB%8C%80%EB%8A%94%2076.4%25%2C%2030%EB%8C%80%EB%8A%94%2066.6%25%EA%B0%80%20%EB%8C%80%EC%A4%91%EA%B5%90%ED%86%B5%EC%9D%84%20%EC%9D%B4%EC%9A%A9%ED%95%98%EA%B3%A0%20%EC%9E%88%EC%97%88%EB%8B%A4.)

**계산 예시:**
```python
지하철_점수 = 0.13 (양재역 근처지만 조금 멈)
버스_점수 = 0.757 (정류장 가까움)

# 정규화 (0~1 범위로 조정)
지하철_정규화 = 0.5  (6개 매물 중 중간)
버스_정규화 = 0.8    (6개 매물 중 상위)

# 통합 점수
교통_점수 = 0.55 × 0.5 + 0.45 × 0.8
         = 0.275 + 0.36
         = 0.635
```

## 4단계: 온도로 변환 (30~43°C)

### 🌡️ 왜 온도로 표현하는가?
- **직관적**: 숫자보다 체감하기 쉬움
- **범위**: 30°C(춥다) ~ 43°C(뜨겁다) = 13도 범위
- **평균**: 36.5°C = 사람 체온 = "딱 적당한" 느낌

```
교통온도 = 30 + (교통 점수 × 13)
```

**온도별 의미:**

| 온도 범위 | 교통 점수 | 의미 | 추천 |
|-----------|-----------|------|------|
| **30-32°C** 🥶 | 0.0-0.15 | 매우 불편 | 차량 필수 |
| **32-34°C** 😰 | 0.15-0.31 | 불편 | 대중교통 불편 |
| **34-36°C** 😐 | 0.31-0.46 | 보통 | 이용 가능 |
| **36-38°C** 🙂 | 0.46-0.62 | 양호 | 편리함 |
| **38-40°C** 😊 | 0.62-0.77 | 우수 | 매우 편리 |
| **40-43°C** 🔥 | 0.77-1.0 | 최우수 |  역세권 |

**계산 예시:**
```python
# 앞의 예시 계속
교통_점수 = 0.635

교통온도 = 30 + (0.635 × 13)
        = 30 + 8.255
        = 38.26°C  🔥 우수 등급!
```

```
매물 위치 (위도, 경도)
    ↓
┌──────────────────┬──────────────────┐
│   지하철 점수    │    버스 점수     │
│   (0~1점)        │    (0~1점)       │
├──────────────────┼──────────────────┤
│ • 거리 (3개역)   │ • 최소 거리      │
│ • 출퇴근 비율    │ • 밀도 (300m)    │
│ • 역 규모        │                  │
└──────────────────┴──────────────────┘
    ↓                    ↓
  정규화             정규화
  (0~1)              (0~1)
    ↓                    ↓
  × 55%              × 45%
    └────────┬────────┘
            ↓
       교통 점수
       (0~1)
            ↓
     × 13 + 30
            ↓
     교통온도 (°C)
   (30~43°C 범위)
```
